# tictactoe-link
https://firebase-tictactoe.herokuapp.com/#/

# Description
## 功能說明
這是個簡易的網頁版井字遊戲，

提供單機遊玩以及支援連線對戰(也可以自己開雙視窗PK)，上限為8人。

連線模式的部分透過 firebase 做即時資料庫處理，專注在盡快完成功能的實踐，

所以 Code 很亂也沒有拆分 Componet 請見諒。
